## 一、注册与账号

### 如何注册账号？
1. 打开优选商城 App 或官网
2. 点击右上角"注册"
3. 输入手机号，获取验证码
4. 设置密码（8-20位，包含字母和数字）
5. 完成实名认证（可选，认证后享更多权益）

### 忘记密码怎么办？
点击登录页"忘记密码"，通过注册手机号接收验证码，重置密码即可。

### 一个手机号可以注册几个账号？
每个手机号只能注册一个账号，建议妥善保管账号信息。

### 账号被封禁怎么处理？
如账号被异常封禁，请拨打客服热线 400-123-4567，提供账号信息和身份证明，客服将在1个工作日内处理。

## 三、下单与支付

### 支持哪些支付方式？
| 支付方式 | 说明 |
|---------|------|
| 微信支付 | 扫码/小程序内直接支付 |
| 支付宝 | 扫码/App内直接支付 |
| 银行卡 | 绑定储蓄卡/信用卡 |

| 花呗/白条 | 支持分期，最高12期免息 |
| 货到付款 | 部分地区支持，仅限特定商品 |

### 如何使用优惠券？
1. 下单后进入"确认订单"页面
2. 点击"优惠券/红包"选项
3. 选择可用的优惠券
4. 优惠券不可叠加使用（会员专属券除外）

### 下单后可以修改地址吗？
- 订单未发货前：进入"我的订单"→ 找到该订单 → 点击"修改地址"
- 已发货后：无法修改，请联系快递员协商改派
- 注意：每笔订单最多修改3次

### 订单可以取消吗？
- 未付款订单：随时可取消，超过30分钟未付款自动取消
- 已付款未发货：进入订单详情点击"申请取消"，商家确认后退款
- 已发货：需申请退货退款

### 可以开发票吗？
支持开具增值税普通发票和增值税专用发票：
- 下单时勾选"需要发票"
- 支持个人抬头和企业抬头
- 电子发票实时到账，纸质发票随货寄出

## 五、售后与退换货

### 退换货政策

| 情况 | 政策 | 时限 |
|------|------|------|
| 不喜欢/不合适 | 7天无理由退货 | 签收后7天内 |
| 商品质量问题 | 30天免费退换货 | 签收后30天内 |
| 功能故障 | 免费维修/换新/退货 | 根据商品保修政策 |
| 收到假货 | 假一赔十 | 签收后30天内 |

### 哪些商品不支持无理由退货？
- 内衣裤、泳衣（卫生原因）
- 定制类商品（如刻字、印花）
- 生鲜食品、拆封化妆品
- 数字商品（充值卡、游戏道具等）
- 已使用或二次销售会影响价值的商品

### 如何申请退款？
1. 进入"我的订单" → 找到该订单
2. 点击"申请售后"
3. 选择退款原因并上传凭证（如有质量问题需拍照）
4. 提交申请后等待商家审核（1-3个工作日）
5. 审核通过后按原路退款

### 退款多久到账？
| 支付方式 | 退款时效 |
|---------|---------|
| 微信支付 | 1-3个工作日 |
| 支付宝 | 1-3个工作日 |
| 银行卡 | 3-7个工作日 |
| 优选余额 | 立即到账 |

## 十、常见问题

### Q1：如何查看我的订单？
进入"我的" → "我的订单"，可按"待付款/待发货/待收货/已完成/退款中"分类查看。

### Q2：收到商品后多久需要确认收货？
系统将在签收后15天自动确认收货。建议检查商品无误后手动确认，确认后货款结算给商家。

### Q3：商品评价有奖励吗？
有！完成评价并上传图片，可获得：
- 普通评价：5积分
- 图文评价：20积分
- 视频评价：50积分
- 优质评价被平台推荐：额外100积分

### Q4：如何分享商品给朋友？
商品详情页点击右上角"分享"，支持微信好友、朋友圈、微博等渠道。朋友通过你的链接下单，你可获得分享红包奖励。

### Q5：平台支持海外购吗？
支持！"全球购"专区提供海外直邮商品，均为官方正品，附有海关清关证明。目前支持美国、日本、韩国、英国、澳大利亚等国家直邮。

### Q6：大型家电如何安装？
大家电（冰箱、洗衣机、空调、电视等）支持送货上门+免费安装服务：
- 下单时选择"送装一体"服务
- 配送人员送货到门并完成安装调试
- 旧机可申请免费上门回收

### Q7：企业采购有哪些优惠？
- 企业认证用户享有专属折扣（最低8折）
- 支持企业对公账户付款
- 可开具增值税专用发票
- 专属企业采购顾问1对1服务
- 联系企业采购热线：400-123-4568

### Q8：商品图片与实物不符怎么办？
如收到商品与图片描述严重不符，可申请仅退款或退货退款，平台将介入处理，保障您的权益。情节严重者，商家将受到相应处罚直至下架。

### Q9：如何投诉商家？
1. 订单详情页 → "投诉商家"
2. 选择投诉原因，填写详情，上传证据
3. 平台将在3个工作日内核实并处理
4. 也可拨打客服热线 400-123-4567

### Q10：积分可以转让给他人吗？
积分不支持转让，仅限本账号使用。但可以使用积分兑换礼品券，将礼品券赠送给他人。
